非遗保护计划

  • 互联网促进计划

    针对非遗领域相关项目、传承人及作品进行新媒体分发及传播，并通过互联网电商服务落地产品转化及提供技术支持。

  • 非遗/青年设计师培训计划

    针对传人进行技艺技能培训，培养年轻技艺传人，延续文化发展；联合设计院校，进行设计师教育及引导，促进文创事业发展。

  • IP保护计划

    打造防伪溯源体系，从创作源头出发，承载作品生产、备案、运输及销售的各个环节，提升和保护非遗文化价值。

  • 公益慈善计划

    建设国际非遗公益慈善基金，以公益之力集全民大众的力量，助力非遗文化发展，保护非遗永久流传。
